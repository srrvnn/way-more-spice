Waymo Spice is a website to label images to help train an ML model to classify spicy food. This is a weekend hack project, so emphasis is on getting things to work, and not necessarily quality of code.

How to use

  • Label images as Food or Spicy on the Train tab.
  • Add new images on the Corpus tab.

    • Labeling Instructions

  • Please label intuitively.
  • If food looks spicy, it probably is.
  • If in doubt, do not mark it spicy.

    • Features

  • Upload images to be labeled, in batches.
  • Preview images before being uploaded.
  • Label images as Food or not, and Spicy or not.
  • Use keyboard shortcuts to label quicker.
  • Use the How to use section for help.
  • Keep track of percentage of corpus trained.

    • Future Work

  • Add food image sets as default.
  • Compress and use EXIF metadata from images when storing on the server.
  • Using tensorflow.js to build a CNN model for the Test tab.
  • Using ml5.js to classify as food to avoid that labeling.

    • Technology Stack

  • Angular - JavaScript view framework
  • Node.js - server side scripting
  • Espress - server framework
  • AWS S3 - object storage to store images
  • PostgreSQL - rel DB to store labels
  • Github - version control hosting frontend code
  • Heroku - cloud service hosting server code
