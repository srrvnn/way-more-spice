Waymo' Spice is a website to help label images to help train an ML model to classify spicy food. This is a weekend hack project, so emphasis is on getting things to work for learning and POC, and therefore not necessarily on the quality of code.

How to use

  • Add new images on the Corpus tab.
  • Train these images on the Train tab.

    • Features

    Current features include:

  • Upload images to be labeled, in batches.
  • Preview images before being uploaded.
  • Label images as Food or not, and Spicy or not.
  • Use keyboard shortcuts to label quicker.
  • Use the How to use section for help.
  • Keep track of percentage of corpus trained.

    • Future Work

  • Use a key-value fast lookup database instead of files.
  • Add Food Image dataset as default.
  • Compress and use EXIF metadata from images when storing on the server.
  • Using tensorflow.js to build a CNN model for the Test tab.
  • Using ml5.js to classify as Food to avoid that labeling.

    • Technology Stack

    The following keep this website together:

  • Angular, frontend code
  • Node.js, backend code
  • Espress, backend REST framework
  • Github, to host frontend code
  • Heroku, to host backend code
